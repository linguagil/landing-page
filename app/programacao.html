Marlon Carvalho

SÉRIO, SUA API NÃO É RESTFUL.

16:10 às 17:00

O desenvolvimento de sistemas monolíticos é passado. No mundo atual, onde os dispositivos móveis estão dominando, não faz mais sentido desenvolver um sistema fechado, que não pode se comunicar e prover dados para dispositivos móveis através de APIs. E, quando o assunto é API, há um monte de confusão. Muitos dizem que sua API é REST. Mas, será que é mesmo? Uma análise superficial em muitas APIs disponibilizadas pela Internet demonstra que não é bem assim. O objetivo desta palestra é demonstrar as ideias por trás do termo RESTful e como elas se aplicam no desenvolvimento de APIs.

PERFORMANCE COM ANGULARJS

14:30 às 15:20

Sua aplicação AngularJS está realmente lenta? Como otimizar sua performance? Combinar outras tecnologias como workaround são paliativos, pois não resolvem o problema.

Entender alguns conceitos do core do AngularJs como o ciclo de vida do $rootScope e como funciona o dirty checking, $compile, $watch, $emit, boas práticas ao utilizar diretivas, filters, services e outros componentes do framework, além da utilização de outros paradigmas e patterns assíncronos são algumas formas abordadas nesta apresentação para melhorar a performance de sua aplicação.
